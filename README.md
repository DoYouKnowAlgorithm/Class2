# 🎉 [DYKA] 알고리즘과 친해지기 [1기]
***
## 🎯 목표
- 작성한 코드의 접근법을 확실하게 말할 수 있다!
- 시간 복잡도를 분석할 수 있다!
- 내 언어와 친해지자!

## 🚴 진행 방식
1. 매주 스터디장이 정한 알고리즘을 공부하고 문제를 풀어온다!
2. 문제는 1주일에 필수(1), 기본(2), 심화(1) 총 4문제를 푼다! **단, 심화는 선택**
3. 스터디날에 랜덤 1명이 공부한 알고리즘을 발표한다!
4. 모든 인원이 각자 한 문제를 선택해 발표한다! ***(접근법 + 시간복잡도 등)***
